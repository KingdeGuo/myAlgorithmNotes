# 插入排序

## 算法描述

前面的元素会先排好序

当前元素与之前已经排好序的元素比较，如果找到合使位置就插入。

## Java

- 升序

  ```java
  public class InsertSort {
      public static int[] insertSort(int[] nums) {
          // 从下标1开始找
          // 因为当只有一个元素时，就是有序的，不需要再排序
          for (int i = 1; i < nums.length; i++) {
              // 记录下来将要排序的数字
              int key = nums[i];
              // 从已排好序的数组最右边开始找
              int index = i;
              // 找到比key小的数，然后插入
              while (index > 0 && key < nums[index - 1]) {
                  nums[index] = nums[index - 1];
                  index--;
              }
              // 如果有比key更小的数，则插入
              if (index != i) {
                  nums[index] = key;
              }
          }
          return nums;
      }
  }
  ```
