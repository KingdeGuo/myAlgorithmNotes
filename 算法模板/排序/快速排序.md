## Java

- 升序排列

```java
// 升序排列
public class BubbleSort {
    public static int[] bubbleSort(int[] nums){
        for (int i = 0; i < nums.length - 1; i++) {
            for (int j = 0; j < nums.length - 1 - i; j++) {
                if (nums[j] > nums[j + 1]) {
                    int t = nums[j];
                    nums[j] = nums[j + 1];
                    nums[j+1] = t;
                }
            }
        }
        return nums;
    }
}
```
