# 选择排序

## 算法描述

前面的是先排好的。

如果是按升序排列，则每次都选出来一个最小的，放到最前面。

需要遍历len-1趟。

## Java

- 升序排序

  ```java
  public class SelectSort {
      public static int[] selectSort(int[] nums){
          for (int i = 0; i < nums.length - 1; i++) {
              int smallIndex = i;
              for (int j = i + 1; j < nums.length; j++) {
                  if (nums[j] < nums[smallIndex]) {
                      smallIndex = j;
                  }
              }
  
              if (i != smallIndex) {
                  int t = nums[i];
                  nums[i] = nums[smallIndex];
                  nums[smallIndex] = t;
              }
          }
          return nums;
      }
  }
  ```
