

```java
class Solution {
    public boolean isPowerOfTwo(int n) {
        return n > 0 && (n & (n - 1)) == 0;
    }
}
```

思路描述：
首先计算出来给定数字n的二进制位数，
然后用1进行左移（注意位数减1）
因为左移的时候右边会用0补齐，
所以用这个数和最开始的数字比较，
如果相等，则说明是2的幂。

```java
class Solution {
    public boolean isPowerOfTwo(int n) {
        if(n < 0){
            return false;
        }
        int tmp1 = n;
        int count = 0;
        while( n >= 1){
            n = n >> 1;
            count++;
        }
        count-=1;
        int tmp2 = 1 << count;
        return tmp1 == tmp2;
    }
}
```

class Solution {
    public boolean isPowerOfTwo(int n) {

        if (n <= 0) {
            return false;
        }
        
        String nums = Integer.toBinaryString(n);
        if (nums.charAt(0) != '1') {
            return false;
        }
        if (nums.length() >= 1) {
            for (int i = 1; i < nums.length(); i++) {
                if (nums.charAt(i) != '0') {
                    return false;
                }
            }
        }
        return true;
    }
}
