```java
class Solution {
    public int climbStairs(int n) {
        if (n == 0 || n == 1) {
            return 1;
        }
        
        int left = 1;
        int right = 1;
        int resu = 0;
        for (int i = 2; i < n+1; i++) {
            resu = left + right;
            left = right;
            right = resu;
        }

        return resu;
    }
    
}
```

变式：
一只青蛙一个可以跳一个台阶，也可以跳两个台阶，......也可以跳n级。
求该青蛙跳上一个n级台阶总共有多少种跳法。
